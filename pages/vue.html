✅ Vue.js (100 Q&A)

🔹 Basics (1-20)

1. What is Vue.js?

Vue.js is a JavaScript framework for building interactive web interfaces and single-page applications.

2. What is the latest major version of Vue.js?

Vue 3 (as of 2025).

3. How do you create a new Vue 3 project using the CLI?

npm init vue@latest

4. What is the data() function used for?

To define reactive variables in a component.

5. What is a component in Vue?

A reusable block of code (template + logic + style).

6. How do you bind data to HTML in Vue?

Using {{ variable }} or v-bind.

7. What is v-model used for?

Two-way binding for form inputs.

8. What is a directive in Vue?

Special tokens in the template starting with v- (e.g., v-if, v-for).

9. What is v-if used for?

To conditionally render an element.

10. What is v-for used for?

To loop through arrays or objects in templates.

11. How do you listen to an event in Vue?

Using v-on or @: Example: @click="doSomething".

12. What is a computed property?

A cached property that recalculates when its dependencies change.

13. What is a method in Vue?

A function defined inside the methods object.

14. What is the difference between computed and methods?

Computed is cached; method runs every time it’s called.

15. How do you add styles scoped to a component?

Use "style scoped" in the component.

16. What does v-bind:class do?

Dynamically bind a class name based on conditions.

17. What does v-bind:style do?

Dynamically apply inline styles.

18. What is a watcher?

A function that watches a reactive variable and runs when it changes.

19. How do you pass data from parent to child?

Using props.

20. How does a child component send data to a parent?

By emitting events with $emit.

🔹 Vue Lifecycle Hooks (21-40)

21. What are lifecycle hooks?

Special functions that run during different stages of a component's life.

22. What is beforeCreate()?

Called before data and events are set up.

23. What is created()?

Called after data and events are set up.

24. What is beforeMount()?

Runs before the component is mounted to the DOM.

25. What is mounted()?

Runs after the component is mounted.

26. What is beforeUpdate()?

Called before the DOM is updated due to reactive data change.

27. What is updated()?

Called after the DOM has been updated.

28. What is beforeUnmount() (Vue 3)?

Runs before a component is removed from the DOM.

29. What is unmounted() (Vue 3)?

Runs after a component has been removed.

30. Which lifecycle hook is best for fetching API data?

mounted()
This is the ideal place to fetch API data since the component is fully mounted and ready.

31. What is setup() in Vue 3?

A new entry point for Composition API logic.

32. Can you use lifecycle hooks inside setup()?

Yes, using onMounted(), onUpdated(), etc.

33. What is onMounted()?

Composition API alternative to mounted().

34. What is onUnmounted()?

Composition API alternative to unmounted().

35. What is the difference between Options API and Composition API?

Options API organizes code by options (data, methods), Composition API organizes by feature logic.

36. What is ref() used for in Vue 3?

To create a reactive variable in Composition API.

37. What is reactive() used for?

To make an object reactive in Composition API.

38. What is toRefs()?

Converts reactive properties into individual ref()s.

39. What is watchEffect()?

A function that automatically re-runs when dependencies change.

40. What is watch()?

A function to manually watch a specific variable for changes.

🧩🔹 Vue Components (41-60)

41. How do you define a component in Vue 3?

With "script setup" or Options API inside .vue files.

42. How do you register a global component?

In main.js:

app.component('MyComponent', MyComponent)

43. What is a prop?

A custom attribute for passing data from parent to child.

44. How do you validate props?

Using

props: { title: String }

45. How to set default props?

With default inside props definition.

46. What is emits in Vue 3?

Defines the events a component can emit.

47. How to emit an event from child to parent?

emit('eventName')

48.How to receive an emitted event in parent?

<ChildComponent @eventName="handler" />

49. How to use slots in Vue?

<slot></slot> in child, content passed from parent.

50. What is a named slot?

Slot with a name:

<slot name="header"></slot>

51. What is a scoped slot?

A slot that passes data from child to parent via props.

52. How to dynamically import a component?

defineAsyncComponent(() => import('./MyComponent.vue'))

53. How to loop over components with v-for?

<Component v-for="item in list" :key="item.id" :data="item" />

54. What is a dynamic component?

Component rendered based on a variable:

<component :is="currentView" />

55. What is keep-alive?

Caches components when switching views.

56. What is v-once?

Renders element once, skips future updates.

57. What is v-html?

Renders raw HTML, use with caution.

58. How to handle form submission in Vue?

<form @submit.prevent="handleSubmit">

59. What is a controlled input?

Input bound with v-model.

60. How to add a custom directive?

app.directive('focus', { mounted(el) { el.focus() } })

🛣️Vue Router & State Management

What is Vue Router?

Official library for managing page routes in Vue.

How to install Vue Router?

npm install vue-router

How to define routes?

In routes.js:

{ path: '/home', component: Home }

How to use dynamic routes?

{ path: '/user/:id', component: User }

How to get route params?

this.$route.params.id

How to navigate programmatically?

this.$router.push('/home')

What is a named route?

A route with a name:

{ name: 'Home', path: '/home' }

How to use router-link?

<router-link to="/home">Home</router-link>

How to set default route?

Use path: '*' or use redirect.

What is Vuex?

State management library (replaced by Pinia in Vue 3).

What is Pinia?

New official state management library in Vue 3.

How to create a Pinia store?

defineStore('main', {
state: () => ({ count: 0 })
})

How to use a store in component?

const store = useMainStore()

How to change state in Pinia?

store.count++
or use actions.

What is an action in Pinia?

Function that changes the store's state.

What is a getter in Pinia?

Computed-like function for state.

Can Pinia stores be split into modules?

Yes, each store is a module.

How to persist Pinia state?

Use pinia-plugin-persistedstate.

Can Pinia work with Composition API?

Yes, fully compatible.

Why choose Pinia over Vuex?

Simpler syntax, better TypeScript support.

🌐API, Axios & Best Practices

What is Axios?

A promise-based HTTP client.

How to install Axios?

npm install axios

How to make a GET request with Axios?

axios.get('/api/data')

How to POST data with Axios?

axios.post('/api/save', { name: 'Vue' })

Where to call APIs in Vue?

Inside mounted() or onMounted().

How to use async/await in Vue?

const data = await axios.get('/api')

How to handle API errors?

Use try...catch.

What is a loading state?

Boolean flag to show loading indicators during API calls.

How to create reusable API functions?

In a separate api.js file.

How to use environment variables in Vue?

Create .env and use

import.meta.env.VITE_API_URL

How to pass props between routes?

Use route props:

props: true

What is a transition in Vue?

Adds animation when element enters/leaves DOM.

How to add a transition?

Wrap element in

<transition></transition>
.

How to debug Vue apps?

Use Vue Devtools in browser.

What is a reactive variable?

A variable that updates the DOM when changed.

What is the difference between ref and reactive?

ref is for primitives; reactive is for objects.

Can you use Composition API in class-style Vue?

No, Composition API is used in <script setup> or normal setup function.

What is teleport in Vue 3?

Moves HTML to a different DOM location:

<teleport to="body">...</teleport>

What is suspense in Vue 3?

Used to show fallback while async component loads.

How to deploy a Vue app?

Run npm run build, then upload the dist/ folder to your server.
