ðŸ”¹ Basics (1â€“20)
1. What is Laravel?
Laravel is a PHP web framework used for building web applications using MVC (Model-View-Controller) architecture.
2. What does MVC stand for?
Model, View, Controller.
3. How do you create a new Laravel project?
composer create-project laravel/laravel project-name
4. What is Artisan in Laravel?
A command-line tool to help with tasks like generating files or running migrations.
5. How to start a local Laravel development server?
php artisan serve
6. What is a route in Laravel?
A URL path that maps to a controller or closure.
7. How do you define a GET route in Laravel?
Route::get('/home', [HomeController::class, 'index']);
8. What is a controller in Laravel?
A class that handles requests and returns responses.
9. How to create a controller?
php artisan make:controller UserController
10. What is a model in Laravel?
A class that represents a database table.
11. How to create a model?
php artisan make:model Product
12. What is migration?
A migration is a file for creating or modifying database tables.
13. How to create a migration?
php artisan make:migration create_products_table
14. How to run migrations?
php artisan migrate
15. How to rollback migrations?
php artisan migrate:rollback
16. What is a middleware?
Middleware filters HTTP requests entering the application.
17. How to create middleware?
php artisan make:middleware CheckAdmin
18. What is CSRF protection?
Protects against cross-site request forgery by verifying a token in forms.
19. How to pass data to a view?
return view('home', ['name' => 'John']);
20. What is the .env file for?
For environment configuration like DB settings.
ðŸ”¹ Eloquent ORM (21-40)
21. What is Eloquent?
Laravel's ORM for interacting with the database using models.
22. How to get all records from a model?
User::all();
23. How to find a record by ID?
User::find(1);
24. How to add a new record?
$user = new User; $user->name = 'John'; $user->save();
25. How to update a record?
$user = User::find(1); $user->name = 'Doe'; $user->save();
26. How to delete a record?
User::destroy(1);
27. What is a relationship?
Links between models (e.g., one-to-many).
28. What is hasMany?
One-to-many relationship.
29. What is belongsTo?
The inverse of hasMany.
30. Example of hasMany:
public function posts() { return $this->hasMany(Post::class); }
31. What is eager loading?
Preloading relationships:
User::with('posts')->get();
32. What is a mutator?
Modify a value before saving.
33. What is an accessor?
Modify a value when getting.
34. What is a migration rollback?
Reverts the last batch of migrations.
35. How to soft delete records?
Use SoftDeletes trait in model.
36. How to get only soft-deleted records?
Model::onlyTrashed()->get();
37. How to restore a soft-deleted record?
Model::withTrashed()->restore();
38. What is $fillable?
List of mass-assignable fields.
39. What is $guarded?
Fields that are not mass assignable.
40. How to use query builder?
DB::table('users')->where('name', 'John')->get();
