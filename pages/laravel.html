1. What is Laravel?
Laravel is a PHP web framework for building web applications using MVC architecture.
2. What does MVC stand for?
Model, View, Controller.
3. How do you create a new Laravel project?
composer create-project laravel/laravel project-name
4. What is Artisan in Laravel?
A command-line tool for tasks like generating files or running migrations.
5. How to start a local Laravel development server?
php artisan serve
6. What is a route in Laravel?
A URL path mapping to a controller or closure.
7. How do you define a GET route in Laravel?
Route::get('/home', [HomeController::class, 'index']);
8. What is a controller in Laravel?
A class handling requests and returning responses.
9. How to create a controller?
php artisan make:controller UserController
10. What is a model in Laravel?
A class representing a database table.
11. How to create a model?
php artisan make:model Product
12. What is migration?
A file for creating or modifying database tables.
13. How to create a migration?
php artisan make:migration create_products_table
14. How to run migrations?
php artisan migrate
15. How to rollback migrations?
php artisan migrate:rollback
16. What is a middleware?
Filters HTTP requests entering the application.
17. How to create middleware?
php artisan make:middleware CheckAdmin
18. What is CSRF protection?
Protects against cross-site request forgery by verifying a token in forms.
19. How to pass data to a view?
return view('home', ['name' => 'John']);
20. What is the .env file for?
For environment configuration like DB settings.
